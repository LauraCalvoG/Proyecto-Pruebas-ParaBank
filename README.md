# Proyecto de Pruebas Automatizadas - ParaBank
## Descripción del proyecto
Este proyecto implementa pruebas automatizadas sobre el sitio [ParaBank](https://parabank.parasoft.com/), una aplicación web bancaria de demostración.  
Las pruebas fueron desarrolladas con **Katalon Studio** y **Cypress**, con el propósito de validar funcionalidades como registro, inicio de sesión, transferencias, actualización de datos y solicitud de préstamos.

## Herramientas y versiones utilizadas
- **Katalon Studio** 9.0.0  
- **Cypress** 13.2.0  
- **Node.js** 20.11.1  
- **Google Chrome** 127+  
- **Sistema operativo:** Windows 10  
- **Editor adicional:** Visual Studio Code
  
##  Requisitos previos
Antes de ejecutar las pruebas asegúrate de tener instalado:
- [Node.js](https://nodejs.org/) (para ejecutar Cypress).  
- [Katalon Studio](https://katalon.com/download) (para las pruebas funcionales).  
- Un navegador compatible (Chrome o Edge).  
- Acceso a internet estable.
